📌 ¿Qué es HTML?

HTML (HyperText Markup Language) es el lenguaje de marcado que se usa para estructurar páginas web.

🎨 ¿Qué es CSS?

CSS (Cascading Style Sheets) es el lenguaje de estilos que permite diseñar y dar formato a las páginas web.

⚡ ¿Qué es JavaScript?

JavaScript es un lenguaje de programación que permite agregar interactividad a las páginas web.
